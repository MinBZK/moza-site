Accounts en tools

Om goed te kunnen samenwerken bij MOZa, heb je toegang nodig tot een aantal tools. Hieronder vind je een overzicht van de belangrijkste systemen.

Mattermost

We gebruiken Mattermost voor dagelijkse communicatie. Hier vind je alle teamleden en kun je vragen stellen, updates delen en meepraten over lopende onderwerpen.

  • Link: digilab.overheid.nl/chat/mijnoverheid-zakelijk (besloten omgeving)
  • Gebruik: Dagelijkse communicatie, vragen, updates
  • Let op: Je hebt een Pleio-account nodig voor toegang (zie de notitie hieronder)
Mattermost instellen

Zorg ervoor dat je Mattermost hebt geïnstalleerd en volg deze stappen:

  1. Registreer een nieuwe account op Pleio
  2. Login (besloten omgeving) → GitLab → Pleio
    • of configureer de volgende server in Mattermost en volg de instructies: https://digilab.overheid.nl/chat
  3. Ga naar het MOZa kanaal (besloten omgeving) of vraag ons om je toe te voegen aan het team

GitHub

We gebruiken GitHub voor code, documentatie en planning. Zoveel mogelijk van onze code werken we open aan. Het is handig om hier rekening mee te houden.

GitHub account aanmaken

GitHub is een openbaar platform. Als je nog geen account hebt:

  1. Maak een account aan op github.com met je rijksoverheid e-mailadres
  2. Kies een gebruikersnaam, waarbij je voor privacy een naam kunt kiezen die niet herleidbaar is naar jouw persoon
  3. Mail OpenSource@MinBZK.nl met de vraag om toegevoegd te worden aan de MinBZK organisatie
  4. Vraag daarna een teamlid om je toe te voegen aan het MOZa team en eventueel aan specifieke repositories

Heb je al een account? Check of zorg dat je rijksoverheid e-mailadres gekoppeld is en volg stap 3 en 4 hierboven.

Repositories

Alle MOZa-repositories hebben de tag mijnoverheidzakelijk:

  • Link: github.com/topics/mijnoverheidzakelijk
  • Gebruik: Code, documentatie, issues

Project boards

We gebruiken GitHub Projects voor planning en overzicht:

  • Alle boards
  • MOZa Fase 3

Op het MOZa Fase 3 bord houden we in hoofdlijnen bij hoe het staat met het plan voor deze fase. Voor gedetailleerde taken zijn er aparte boards voor UX, Development, Communicatie, Architectuur en Research.

Samenwerkruimte

De Samenwerkruimte is onze centrale plek voor programmadocumentatie, presentaties, verslagen en andere documenten.

  • Link: samenwerkruimten.nl MOZa (besloten omgeving)
  • Gebruik: Programmadocumentatie, presentaties, verslagen en overige documenten

Hier vind je onder andere:

  • Verslagen van de Stuurgroep en het Regieteam
  • Presentaties en roadmaps
  • Het smoelenboek met contactgegevens
  • De contentkalender

Andere tools

Afhankelijk van je rol kun je ook toegang nodig hebben tot:

  • Figma - Voor UX/design work
  • Lucidspark - Voor brainstormen en visueel samenwerken

Vraag een collega welke tools voor jouw werk relevant kunnen zijn.

Hulp nodig?

Als je ergens geen toegang toe hebt of problemen ondervindt, vraag het in Mattermost of tijdens een samenwerkdag.
